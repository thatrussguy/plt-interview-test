## Set up a copy to try out locally

### Prerequisites

- [Git](https://git-scm.com/)
- [Node](https://nodejs.org)

### Instructions

Clone this repo

```bash
git clone https://github.com/thatrussguy/plt-interview-test.git
```

Open project folder

```bash
cd plt-interview-test
```

Install dependencies

```bash
npm install
```

Start the app locally

```bash
npm start
```
